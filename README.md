# MentorBridge

MentorBridge is a frontend-only career mentorship demo platform built with React, JavaScript, HTML, and CSS.

## Stack

- React
- JavaScript
- HTML
- CSS
- Browser localStorage for demo data

## Features

- Mentor matching system
- Career guidance sessions
- Resume review with local suggestions
- Progress tracking
- Mentor rating system
- Community Q&A forum
- Analytics dashboard

## Project Structure

```text
apps/
  web/   React frontend
```

## Setup

1. Install dependencies:

```bash
npm install
```

2. Run the app:

```bash
npm run dev
```

3. Open the homepage and click `Reset Demo Data` if you want to restore the local demo dataset.

## Notes

- Everything runs in the frontend only.
- Demo accounts, sessions, progress, resume reviews, and forum content are stored in browser localStorage.
- Resetting demo data restores the default local dataset.
